Restauración de una imagen de forma remota

Anteriormente, ya hemos usado Clonezilla para restaurar una imagen de forma local. Sin embargo, también tenemos la opción de hacerlo de forma remota haciendo uso de servidores comom ssh o samba server. Hoy aprenderemos a usar Clonezilla usando estos servidores

Lo primero que haremos en este tutorial será crear este USB usando el programa BalenaEtcher para Ubuntu, al igual que hicimos en el anterior tutorial

Clonación de la imagen

-Restauración remota: La restauración remota implica la restauración de una imagen desde un servidor remoto utilizando una conexión de red. Esto puede ser útil si necesitas restaurar varios ordenadores en una red o si necesitas restaurar una imagen en una ubicación remota. Para la restauración remota, debes configurar Clonezilla para acceder al servidor remoto y seleccionar la imagen que deseas restaurar. Es importante tener una conexión estable y de alta velocidad para garantizar que la imagen se restaure correctamente.
